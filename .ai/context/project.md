# Project Context

## Project Name

AI Telecaller for Hospitals (MVP)

---

## Overview

AI Telecaller is an AI-powered hospital assistant that answers patient questions using Retrieval-Augmented Generation (RAG). It provides accurate information from hospital knowledge sources and escalates conversations to human staff when necessary.

The long-term vision is to evolve into:

- AI Voice Telecaller
- AI Agent Platform
- Multi-Tenant SaaS

Current development focuses exclusively on **Phase 1 (MVP)**.

---

## Current Scope

The MVP includes:

- Website AI Chatbot
- RAG-based question answering
- Admin Dashboard
- Authentication
- Hospital Knowledge Base
- Doctor Management
- Department Management
- FAQ Management
- Conversation History
- Human Escalation

Do not implement future roadmap features unless explicitly requested.

---

## Primary Goals

- Accurate hospital information
- Zero hallucination for supported knowledge
- Secure architecture
- Production-ready code
- Modular design
- Easy future expansion

---

## Product Principles

- Retrieval before generation
- Human safety first
- Security by default
- Maintainability over complexity
- Simplicity over premature optimization
