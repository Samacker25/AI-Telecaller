# Refactoring Workflow

1. Understand the existing implementation.
2. Ensure adequate test coverage.
3. Refactor in small, incremental steps.
4. Preserve external behavior.
5. Remove duplicated code.
6. Improve readability and maintainability.
7. Run all tests.
8. Verify performance has not degraded.
9. Update documentation if necessary.
10. Complete Pull Request Checklist.
